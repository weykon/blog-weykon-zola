「linux」ssh 免登录

2022-11-23T17:31:28Z

只留下流程

对象：

- 服务器 A 
- 电脑 B

在电脑B操作的

  1. 生成钥匙对
ssh-keygen -t rsa
  1. 设置权限
chmod 600 <which is *.pub\>
  1. 提交写入 (提交公钥至服务器 A)
ssh-copy-id -i <*.pub> user@server
  1. （可选）设置电脑 B ssh配置

编辑 ~/.ssh/config

添加

Host <快速键入的名称>
    HostNmae <服务器A地址>
    User <登陆服务器A用户名>
    IdentityFile <上面生成的两个文件中，没有后缀名的那个>

保存后运行

ssh <快速键入的名称>
# 例如 ssh mysite
