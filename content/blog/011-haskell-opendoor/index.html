软件构造

GHC 是 Haskell 的软件管理工具。

要使用 Haskell 语言构建软件，我们在这个软件的入口理解，先使用把 GHC 好好利用起来。

可以直接跳到 GHC 的安装说明 https://www.haskell.org/ghcup/

Mac、 Linux 可直接， windows 另看说明。

curl --proto '=https' --tlsv1.2 -sSf https://get-ghcup.haskell.org | sh

这条命令目前有时候会在一些步骤卡住

  • 设置 cabal 的config 卡住
  • 从 某个地址 下载由于网络原因(agent)会卡住

不过这个时候即使安装程序卡住也没问题, ctrl + c 退出来。

使用 ghcup tui 命令，会显示可下载、推荐安装的工具。 把他们都下载下来就好。
