局域网可接入

这个功能能解决一些设备中具有网络代理设置的代理问题 接入设备中进入网络配置代理设置，地址的设置是提供代理的设备并连接到提供服务的端口 对于移动设备大部分都具有这个设置 但是XBOX上没有，这个时候无法设置代理配置，就要直接连接一个具有代理转发的设备来实现 就是下面的DCHP功能，也具有可玩性

DCHP的功能

首先我让机器设置为可代理转发的状态，方法

在代理XBOX的时候，我是成功了代理一定的服务，但是在战区的游戏对战的服务，我的代理失败了。 这种情况只好用游戏加速器了。😭

而对于我使用Quest2的代理（Quest2的系统限制很容易导致设备中的代理失效，比如时间会错而无法设置） 我使用DCHP的功能给他转发，但是q2设置直接连接回了路由器作网关ip了似乎不成功 这个时候我只会使用普通的系统代理，然后连接wifi，在sidequset中使用ADB控制直接加入代理环境的ip和port adb set proxy

这个功能是出乎我意料，不过连接的时候用起来比较麻烦。 在Clash for windows当中

  1. 确保TUN MODE正确运行
  2. 确保路由器是关闭了DCHP功能
  3. 让想要有代理的设备重新连接局域网
  4. 见到新设备进入后，点击切换配置启动代理（设置好像又要重新进入局域网）

在macbook里控制是否代理转发， 0: 关闭， 1: 开启。

sudo sysctl -w net.inet.ip.forwarding=0
