「linux」source command

2022-11-13T17:55:29Z

The source command is a built-in shell command used to read and execute commands from files in the current shell session. The source command is typically used to preserve and change environment variables in the current shell. In short, source is a script that runs the execute command in the current shell.

(source命令是一个内置的shell命令，用于从当前shell会话中的文件读取和执行命令。source命令通常用于保留、更改当前shell中的环境变量。简而言之，source一个脚本，将会在当前shell中运行execute命令。)

The source command can be used to:

  • Refresh the current shell environment
  • Execute Shell scripts in the current environment using source
  • Import a Shell function from the script in the environment
  • Reads variables from another Shell script

(source命令可用于：

  • 刷新当前的shell环境
  • 在当前环境使用source执行Shell脚本
  • 从脚本中导入环境中一个Shell函数
  • 从另一个Shell脚本中读取变量)
