「git-push-url」推送多个仓库

2022-08-01T10:20:05Z

当我们有代码推送到多个仓库：

  • 多个remote （常用的是： git remote add origin <git_repo_url>）

  • 同一个remote下多个url （类似的是： git remote set-url origin <git_repo_url>）

在 "set-url" 无法做到添加url，而是替换url。

这个工具就为了简单添加 url ，没错就是这么简单。

用途

推送多个仓库

用法

npm i git-push-url -g
git-push-url add https://github.com/user/repo.git
#or
git-push-url remove https://github.com/user/repo.git

原理

改变了一下当前目录的 "./.git/config" 文件里面的 remote 内容，添加 url 或 删除 url。
